# ☕ Cosy Café Website Template

## 📌 Overview
A warm and inviting website template designed for independent cafés that want to reflect a relaxed, homely atmosphere online.

This project focuses on soft colour palettes, comfortable spacing, natural textures, and welcoming typography to create a digital experience that mirrors the feeling of stepping into a cosy neighbourhood café.

The aim was to explore how thoughtful design choices can communicate warmth, comfort, and authenticity while maintaining clarity and usability.

---

## 🎨 Design Direction

This template embraces:
- Warm beige, cream, and soft brown tones  
- Comfortable spacing and breathable layouts  
- Friendly, readable typography  
- Soft visual hierarchy  
- Subtle hover interactions for a relaxed feel  

The overall aesthetic is calm, intimate, and community-focused.

---

## ✨ Key Features

- Fully responsive layout (mobile-first approach)
- Clear, easy-to-navigate menu section
- About section designed to tell the café’s story
- Image-focused layout to highlight ambience and products
- Contact section with clear call-to-action

---

## 🛠 Tech Stack

- HTML  
- CSS  
- Bootstrap  

---

## 🧠 Project Focus

This project was created to practise:
- Translating brand personality into UI design  
- Creating inviting layouts through spacing and typography  
- Designing for small, independent businesses  
- Maintaining simplicity without sacrificing visual warmth
